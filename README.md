# AI Utility Toolkit

This is a collection of simple Python utility scripts that are useful for anyone starting out with AI development.

## Features
- **Text Cleaner**: A script to clean up and normalize text data.
- **Text to Speech**: A simple utility to convert text to speech.
- **Chatbot**: A basic conversational AI powered by machine learning.
